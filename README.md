## Архітектура програмних класів

### 1. Клас Artist (Виконавець)
* `string Name` — Назва групи або виконавця.
* `string Genre` — Основний музичний жанр.

### 2. Клас Song (Пісня)
* `string Title` — Назва композиції.
* `Artist Performer` — Посилання на об'єкт класу Artist.

### 3. Клас Album (Диск)
* `string AlbumTitle` — Назва диска.
* `List<Song> Tracks` — Список посилань на об'єкти пісень.

### 4. Клас MusicLibrary (Логіка)
* `FindSongsByArtist()` — Пошук усіх пісень заданої групи.
* `FindAlbumsBySong()` — Пошук дисків, де є конкретна пісня.
