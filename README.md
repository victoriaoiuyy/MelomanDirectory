## Архітектура програмних класів
Згідно з індивідуальним завданням, програма базується на системі об'єктних посилань:
* **Artist** (Виконавець) — базовий клас для зберігання даних про музикантів.
* **Song** (Пісня) — містить назву та посилання на відповідний об'єкт класу Artist.
* **Album** (Диск) — містить назву та динамічний список посилань на об'єкти пісень.

---

## Інструкція користувача
Програма реалізує наступні ключові функції:
1. Вибір усіх пісень заданої групи/виконавця.
2. Пошук усіх дисків (альбомів), до складу яких входить обрана пісня.
